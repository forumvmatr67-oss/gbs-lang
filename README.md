## ⚡ Что это?

GBS - это простой язык программирования для быстрых скриптов на Windows

## 📦 Установка

```bash
git clone https://github.com/yourname/gbs-lang.git
cd gbs-lang
install.bat
```

## 🎮 Пример кода

```gbs
# Приветствие
print "What's your name?"
io.read()
set name = %GBS_INPUT%
print "Hello, "
print name

# Калькулятор
set x = 10
set y = 20
set sum = x + y
print "Sum: "
print sum
```
